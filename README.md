📊 Learning Strategy Comparison Tool

An interactive web-based application that compares Rote Learning and Inductive Learning using a real-world diabetes dataset.

This project demonstrates how inductive learning generalizes better than memorization-based approaches through predictions and visualizations.

🚀 Features
📄 Dataset preview
🧠 Comparison of Rote vs Inductive Learning
📊 Feature importance visualization
🔍 Manual input for prediction
⚖️ Side-by-side prediction comparison
📉 Graph-based result analysis

🧪 Technologies Used
Python
Streamlit
pandas
scikit-learn
matplotlib

⚙️ How to Run Locally

Clone the repository:
git clone https://github.com/unnatidharpure13-pixel/Learning-Strategy-Comparison-Tool.git

Navigate to the folder:
cd learning-strategy-comparison

Install dependencies:
pip install -r requirements.txt

Run the app:
python -m streamlit run app.py

📊 Learning Strategies Explained

🔹 Rote Learning
Memorizes training data
Works only for exact matches
Cannot generalize

🔹 Inductive Learning
Learns patterns from data
Predicts unseen inputs
Used in real-world AI systems

📈 Key Insights
Inductive Learning performs significantly better in real-world scenarios
Rote Learning is limited to memorization tasks
Features like Glucose and BMI strongly influence predictions

🎯 Applications
Healthcare prediction systems
Fraud detection
Recommendation systems
Data-driven decision making
