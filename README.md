# 📊 Learning Strategy Comparison Tool

An interactive web-based application that compares **Rote Learning** and **Inductive Learning** using a diabetes dataset.

---

## 🌐 Live Demo

👉 https://learning-strategy-comparison-tool.streamlit.app/

---

## 🚀 Features

- Dataset preview  
- Comparison of Rote vs Inductive Learning  
- Feature importance visualization  
- Manual input for prediction  
- Side-by-side prediction comparison  
- Graph-based result analysis  

---

## 🧠 Learning Strategies Explained

### 🔹 Rote Learning
- Memorizes training data  
- Works only for exact matches  
- Cannot generalize  

### 🔹 Inductive Learning
- Learns patterns from data  
- Predicts unseen inputs  
- Used in real-world AI systems  

---

## 📈 Key Insights

- Inductive Learning performs better in real-world scenarios  
- Rote Learning is limited to memorization  
- Features like Glucose and BMI are important
